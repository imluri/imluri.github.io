      How to Connect
    

    
  

  

    

      
      You're viewing this page over HTTPS. Browsers block an HTTPS page from reaching a local http://localhost bridge, so connecting here won't work. Run the UI from a local http:// origin instead: download the site repo, run python serve.py, and open http://localhost:5500/iyai. Then follow the steps below.
    

    
    
      
  1. 
        1
        
    
          Install the IYAI plugin into Infinite Yield
          
    Download the plugin files from the IYAI GitHub repository and place them in your executor's workspace/ folder. Inject your executor, open Infinite Yield, and run plugin iyai to install it. Then open the UI anytime with ;ai. Configure your API key under its Settings tab.
    
        
    
      
    2. 
      
  2. 
        2
        
    
          Start the local bridge server
          
    Run iyai_bridge.py (requires Python + Flask) or iyai_bridge.ps1 (PowerShell, no dependencies) from the repo. The bridge listens on port 7402 and relays messages between this page and the plugin.
    
        
    
      
    3. 
      
  3. 
        3
        
    
          Enable the bridge in IYAI
          
    In the Roblox plugin, go to the Browser page and click Connect to Browser. The plugin will begin polling the bridge for incoming messages.
    
        
    
      
    4. 
      
  4. 
        4
        
    
          Connect this page
          
    Leave the URL as http://localhost:7402 and click Connect. Settings, history, and tools will sync automatically from the plugin.
    
        
    
      
    5. 
    

    

      
      Everything runs locally — the bridge only accepts connections from localhost and nothing is sent to any external server.
