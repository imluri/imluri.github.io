How to Use
  1. Load a .glb or .gltf file above
  2. Switch to Orbit — drag to rotate, scroll to zoom
  3. Switch to Place Pin and click any surface, or pick a part from Model Parts below
  4. Give the pin a label and optional description
  5. Repeat for all annotation points
  6. Click Export JSON to download pin data
